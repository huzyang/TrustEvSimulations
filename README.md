# TrustEvSimulations

## How to run

git clone 

### creat venv
conda create --name TrustEvSimulations python=3.11 -y
or
conda create --prefix=D:\Anaconda\envs\TrustEvSimulations python=3.11 -y

conda activate TrustEvSimulations

### install packages

```python
# Installing HyperNetX
pip install hypernetx

# Install Mesa
# To install our latest stable release (2.4.x), run:
pip install -U mesa[rec]

# To install our latest pre-release (3.0.0 alpha), run:
pip install -U --pre mesa[rec]

# Install Jupyter notebook (optional非必选):
pip install jupyter

# Install Seaborn (which is used for data visualization in Mesa):
pip install seaborn
```
